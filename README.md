A **fun and interactive game** in Python where you play **Rock, Paper, Scissors** against the computer.
This project is beginner-friendly and demonstrates **user input, random choices, conditionals, and ASCII art**.

---

## Features

* Play against the computer
* Interactive **ASCII art** for Rock, Paper, and Scissors
* Handles invalid inputs
* Determines the winner using simple game logic
* Shows a draw when both choices are the same

---

## How to Play

1. Run the program:

```bash
python rock_paper_scissors.py
```

2. Enter your choice:

   * `0` ‚Üí Rock
   * `1` ‚Üí Paper
   * `2` ‚Üí Scissors

3. The program displays your choice and the computer‚Äôs choice using ASCII art.

4. It announces the winner, loser, or a draw.
